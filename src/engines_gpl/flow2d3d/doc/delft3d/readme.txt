VERSION NUMBER
--------------

The version number of Delft3D-FLOW can be found in the file:

    include\flow\sysdef.gdt 

The version number of Delft3D-MOR can be found in the file:

    progsrc\mor\version.dat


BUILDING
--------

To build Delft3D on Linux systems, type:

    . ./setup wlinux
    make

To build Delft3D on Windows with Visual Studio:

    open the relevant .dsw file in the developer
    press F7
